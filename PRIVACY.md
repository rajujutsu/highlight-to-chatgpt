# Privacy Policy

## Highlight → Ask ChatGPT

This Chrome extension does not collect, store, or transmit personal data.

## Data Usage
The extension processes text **only when a user explicitly selects text and chooses to send it to ChatGPT** using the right-click context menu.

- The selected text is used solely to populate the ChatGPT input field.
- All processing occurs locally within the user’s browser.
- No data is sent to the developer or to any external servers.

## Data Collection
This extension does **not** collect:
- Personally identifiable information
- Authentication credentials
- Financial or payment information
- Location data
- Browsing history
- User activity analytics

## Data Sharing
The extension does not sell, transfer, or share user data with third parties.
