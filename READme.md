# Highlight → Ask ChatGPT via Google Extension

Highlight text on any webpage, right-click, and open a new ChatGPT tab with the selected text automatically inserted (ready to send).

## Features
- Right-click context menu on selected text
- Opens a new ChatGPT tab every time
- Automatically inserts selected text into ChatGPT
- No API keys or setup required

## How to Use
1. Highlight any text on a webpage
2. Right-click → “Ask ChatGPT”
3. A new ChatGPT tab opens with your text inserted

## Permissions
- contextMenus – adds the right-click menu
- tabs – opens a new ChatGPT tab
- scripting – inserts the selected text into ChatGPT

Host permissions are limited to:
- https://chatgpt.com/*
- https://chat.openai.com/*

## Privacy
This extension does not collect, store, or transmit personal data.
It only processes text explicitly selected by the user.
